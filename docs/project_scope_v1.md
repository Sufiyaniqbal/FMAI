# FileMaker AI Analysis – Version 1 Scope

Version 1 will:

- Accept CSV file exported from FileMaker
- Read the data using Python
- Allow user to enter a simple text prompt
- Generate basic reports such as:
  - Total sales
  - Sales by customer
  - Sales by month
- Output:
  - CSV report
  - Simple explanation text

Version 1 will NOT:

- Modify FileMaker database
- Connect directly to FileMaker Server
- Analyze schema or scripts
- Use live production data
